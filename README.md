# Telegram Bot для учета работы

Умный AI бот для текстильной компании с полной базой данных.

## Возможности

✅ AI обработка с уточнениями
✅ База данных Supabase
✅ Команды /stats, /report, /help
✅ Уведомления шефу
✅ Работает 24/7

## Установка локально

1. Установи Node.js (если нет)
2. Скачай все файлы
3. Запусти:
```bash
npm install
npm start
```

## Развертывание на Railway

1. Создай аккаунт на railway.app
2. New Project → Deploy from GitHub
3. Добавь переменные окружения из .env
4. Deploy!

## Переменные окружения

```
TELEGRAM_BOT_TOKEN=...
SUPABASE_URL=...
SUPABASE_KEY=...
OPENAI_API_KEY=...
BOSS_TELEGRAM_ID=...
```

## Команды бота

- `/start` - начало
- `/stats` - статистика
- `/report` - отчет за день
- `/help` - помощь

## Для сотрудников

Просто пиши в свободной форме:
- "Принял заказ 50 боди"
- "Отгрузил Анне 100 штук"

Бот сам поймет и уточнит детали!
